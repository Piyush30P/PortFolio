# Piyush Pise — AI & Full-Stack Engineer

A modern, interactive portfolio website showcasing my work in **AI**, **full-stack development**, and **scalable product engineering**.

## About

This portfolio highlights:
- Professional experience in software engineering and AI-driven development
- Technical skills across frontend, backend, cloud, and data systems
- Selected projects with impact-focused outcomes
- Publications, achievements, and contact information

The site is designed with a futuristic dark UI, smooth animations, interactive project cards, and responsive layouts for mobile and desktop.

## Tech Stack

- **HTML5**
- **CSS3** (custom properties, responsive design, animations)
- **JavaScript (Vanilla)**
- **Font Awesome**
- **Google Fonts** (Space Grotesk, Inter, JetBrains Mono)

## Features

- Scroll progress indicator
- Custom cursor interactions (desktop)
- Animated hero section with typewriter-style role highlight
- Skills filtering UI
- Timeline-based experience section
- Project cards with modal details
- Publications and achievements showcase
- Responsive navigation with mobile menu

1. Clone the repository:

2. Open the project folder.
3. Run the site by opening `index.html` in your browser.
